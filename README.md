# cyber-dashboard-scheduler

## Démarrage rapide

1. Créer un environnement virtuel Python
2. Installer les dépendances avec `pip install -r requirements.txt`
3. Copier `.env.example` vers `.env` puis renseigner les variables
4. Lancer le point d'entrée minimal avec `python -m cyber_dashboard_scheduler.main`

## Structure initiale

```text
cyber_dashboard_scheduler/
├── clients/
├── config/
├── db/
├── models/
├── repositories/
├── services/
├── utils/
└── main.py
```
